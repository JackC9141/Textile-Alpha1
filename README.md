# Textile-Alpha1

App Name: Textile 

App Type: Social Media App 

Problem It Solves: The problem with current social media apps is they miss something that would improve user experience. For example, Discord lacks a search function whilst Reddit has a powerful search function, but Reddit lacks Discord’s architecture for user interaction. Textile integrates the good parts of mainstream social media apps in a seamless and user-friendly way, blurring the lines between forums/threads and live chats. 

 

Page 1 — Name & Description 

Home (index.html): the landing page for Textile. It greets users with the app name and then provides quick navigation cards to the three core sections (profile, forums, posts). 


Page 2 — Name & Description 

Forums: You can browse the forums you joined (similar to Discord servers), explore the threads inside of that forum (similar to Discord channels), and then chat live inside each thread. 


Page 3 — Name & Description 

Posts: Displays which posts you have made on which threads of which forums. 


Page 4 — Name & Description (optional) 

Profile: You can customize your avatar, name, handle, stats, adjust privacy settings that show whether to hide or show your posts and to whom, etc. 


Key Interactive Feature 1 

Burger menu navigation: useful for people on mobile to jump between Home/Forums/Profile/Posts/Settings/Help without losing their place. 


Key Interactive Feature 2 

Forum & Thread Browsing: users can browse a list of forums they belong to, see their threads, and participate in the chat inside a thread. 


Key Interactive Feature 3 

Profile Tab Switching: users can switch between posts, forums, and servers to view different aspects of their activity without switching pages. 


Key Interactive Feature 4 

Search function: unlike Discord or stoat, Textile is powered with a search function to search for forums and—in the possible future—user posts.
